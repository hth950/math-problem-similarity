# 수학 문제 난이도 판정 기준서

## 1. 문서 개요

### 목적

이 문서는 LLM이 수학 문제의 난이도를 1~5등급으로 자동 판정하기 위한 기준을 제공합니다. 본 기준은 프롬프트의 핵심 참조 자료로 사용되며, LLM이 문제 정보(`question`, `refer`, `choice1~5`, `school_level`, `grade`)를 입력받아 일관되고 재현 가능한 방식으로 난이도를 판정하는 데 목적이 있습니다.

### 대상 데이터 설명

| 항목 | 내용 |
|------|------|
| 총 문제 수 | 약 100,000문제 |
| school_level | middle(중학교), high(고등학교), elementary(초등), unknown |
| grade | 1, 2, 3, None |
| type | multiple_choice(객관식), short_answer(단답형), essay(서술형) |
| 주요 출처 | 내신 기출(중간/기말), 교재(쎈, RPM, 마플시너지, 올림포스 등) |
| 난이도 필드 | `level` (1~5 정수, 없으면 None) |

**기존 level 분포 참고 (재판정 전):**
- level=1: 21,948문제 (21.9%)
- level=2: 21,206문제 (21.2%)
- level=3: 34,378문제 (34.4%) — 최다 비중
- level=4: 1,720문제 (1.7%)
- level=5: 570문제 (0.6%)
- None: 20,178문제 (20.2%)

## 2. 난이도 등급 정의

### Level 1 — 기초 (Basic)

| 항목 | 기준 |
|------|------|
| **등급명** | 기초 |
| **한줄 정의** | 공식·정의를 직접 대입하거나 단순 계산으로 풀 수 있는 문제 |
| **행동영역** | 계산(Computation) / 지식 확인(Knowledge Recall) |
| **개념 연계 수** | 1개 개념, 단일 단원 |
| **풀이 단계 수** | 1~2단계 |
| **계산 복잡도** | 단순 사칙연산 또는 공식 1회 대입 |
| **정답률 기대치** | 85% 이상 |
| **대응 수준** | 개념기본서 기본예제, 수능 2점 문항 수준 |

**중학교 예시 특성:**
- 정의를 그대로 적용: "다음 중 소수인 것은?", 약수의 개수 구하기
- 기본 공식 대입: 일차방정식 풀기(이항 1회), 기본 도형 넓이·둘레
- 특정 값 대입 후 계산: 함수 `f(x)=2x+1`에서 `f(3)` 구하기

**고등학교 예시 특성:**
- 정의 직접 확인: 포물선 초점 좌표 구하기, 집합의 원소 나열
- 단순 공식 대입: `sin30°` 값, 등차수열 일반항 `a_n = a_1 + (n-1)d` 대입
- 단항식 미분: `f(x)=x³` → `f'(x)` 구하기

### Level 2 — 기본 (Elementary)

| 항목 | 기준 |
|------|------|
| **등급명** | 기본 |
| **한줄 정의** | 기본 유형을 인식하고 표준 풀이 절차를 적용하는 문제 |
| **행동영역** | 이해(Comprehension) / 기본 적용(Basic Application) |
| **개념 연계 수** | 1~2개 개념, 주로 단일 단원 내 |
| **풀이 단계 수** | 2~3단계 |
| **계산 복잡도** | 이항·통분·인수분해 등 중간 수준 계산 포함 |
| **정답률 기대치** | 65~85% |
| **대응 수준** | 쎈 A단계, RPM 기본유형, 수능 3점 하위권 문항 |

**중학교 예시 특성:**
- 연립방정식 풀기(가감법/대입법 1회 적용)
- 이차방정식 인수분해로 풀기
- 삼각형의 넓이(피타고라스 정리 1회 활용)
- 기본 확률 계산: 전체 경우의 수 / 특정 경우의 수

**고등학교 예시 특성:**
- 순열·조합 공식 적용 (단순 경우 수 계산)
- 로그 기본 성질 적용: `log_2 8` 계산, 지수·로그 변환
- 등비수열 합 공식 직접 적용
- 이차함수 꼭짓점·축 구하기

### Level 3 — 표준 (Standard)

| 항목 | 기준 |
|------|------|
| **등급명** | 표준 |
| **한줄 정의** | 2~3개 개념을 결합하여 표준 유형 문제를 풀되, 약간의 응용·변형이 필요한 문제 |
| **행동영역** | 적용(Application) / 분석(Analysis) 초입 |
| **개념 연계 수** | 2~3개 개념, 단원 내 혹은 인접 단원 간 |
| **풀이 단계 수** | 3~5단계 |
| **계산 복잡도** | 복수의 연산이 연결되며, 조건 해석 필요 |
| **정답률 기대치** | 40~65% |
| **대응 수준** | 쎈 B단계, RPM 발전유형, 수능 3점 상위 ~ 4점 하위 |

**중학교 예시 특성:**
- 이차방정식 + 도형: 직사각형의 가로/세로를 변수로 놓고 이차방정식 세우기
- 연립방정식 활용문제: 속도·거리·시간, 농도 혼합
- 도형의 닮음 + 넓이 비: 닮음비를 이용해 넓이 구하기
- 통계: 평균·분산 동시 활용

**고등학교 예시 특성:**
- 등차·등비수열 조건 동시 적용
- 벡터 등호 조건 세우고 성분 비교
- 이차함수 + 부등식: 모든 실수에서 성립하는 조건
- 삼각함수 기본 방정식 풀기(범위 내 해 구하기)

> **주의:** 전체 문제의 약 34%가 이 레벨에 집중되어 있어, 경계 판정 시 level=2와 level=4 사이를 명확히 구분해야 합니다.

---

### Level 4 — 심화 (Advanced)

| 항목 | 기준 |
|------|------|
| **등급명** | 심화 |
| **한줄 정의** | 복합 개념을 연결하고 다단계 추론이 필요한 문제 |
| **행동영역** | 분석(Analysis) / 종합(Synthesis) |
| **개념 연계 수** | 3~4개 이상 개념, 단원 간 연계 |
| **풀이 단계 수** | 5~8단계 |
| **계산 복잡도** | 복잡한 식 변환, 조건 설정 후 역추적 포함 |
| **정답률 기대치** | 15~40% |
| **대응 수준** | 쎈 C단계, 마플시너지·일품 발전문제, 수능 4점 문항 |

**중학교 예시 특성:**
- 직선의 방정식 + 넓이 이등분: 조건이 2개 이상 결합
- 이차함수 + 넓이의 최댓값·최솟값(미적분 없이)
- 증명 문제: 삼각형 합동·닮음 2단계 이상 논증
- 경우의 수 + 확률 + 조건 해석 복합

**고등학교 예시 특성:**
- 두 원의 교점과 제3의 원 사이 관계
- 적분 + 넓이 + 최솟값: `f(t) = ∫ |g(x)| dx` 형태
- 행렬·벡터 + 추가 조건 설정 (수능 4점 하위)
- 삼각함수 + 도형 + 넓이: 여러 공식 동시 적용

### Level 5 — 최고난도 (Expert)

| 항목 | 기준 |
|------|------|
| **등급명** | 최고난도 |
| **한줄 정의** | 창의적 접근·고차원 추론이 요구되며, 보기 판별·증명·아이디어 발상이 핵심인 문제 |
| **행동영역** | 추론(Reasoning) / 창의·융합(Creative Synthesis) |
| **개념 연계 수** | 4개 이상, 단원 간 고도 복합 연계 |
| **풀이 단계 수** | 8단계 이상 혹은 풀이 경로 자체를 발견해야 하는 문제 |
| **계산 복잡도** | 최고 수준, 풀이 방향 설정 자체가 비자명 |
| **정답률 기대치** | 15% 미만 (수능 킬러는 1~5%) |
| **대응 수준** | 블랙라벨·최상위, 수능 킬러(21·29·30번), 올림피아드 유형 |

**중학교 예시 특성:**
- 수학경시대회 수준 문제: 여러 단원 복합 + 증명
- 규칙 발견 + 일반화 (귀납적 추론 필요)
- 창의적 보조선 작도 후 증명

**고등학교 예시 특성:**
- 로그 부등식 + 함수 개수 판별 (보기 T/F 각각 판단)
- 삼차함수 조건 분석: 근의 부호·개수 + 계수 조건 동시 추론
- 미적분 + 도형 + 최솟값의 최솟값 (2중 최적화)
- 수열 + 점화식 + 귀납 증명 복합

## 3. 난이도 판정 기준표

아래 표는 LLM이 단계적으로 점수를 매겨 최종 레벨을 도출하는 데 사용합니다.

### 3-1. 행동영역 점수 (A)

| 행동영역 | 판별 기준 | 점수 |
|----------|-----------|------|
| 계산/지식 확인 | 공식에 수를 대입하거나 정의를 그대로 확인 | 1 |
| 이해/기본 적용 | 개념을 이해하고 표준 절차를 따라 풀기 | 2 |
| 적용 | 알려진 유형이지만 조건 해석·변형 필요 | 3 |
| 분석/종합 | 여러 개념 연결, 중간 결과를 활용해 다음 단계 도출 | 4 |
| 추론/창의 | 풀이 경로 자체를 발견해야 하거나 고차원 논증 필요 | 5 |

### 3-2. 개념 연계 수 점수 (B)

| 연계 개념 수 | 점수 |
|-------------|------|
| 1개 (단일 개념, 단일 단원) | 1 |
| 2개 (단일 단원 내 결합) | 2 |
| 3개 (인접 단원 간 연계) | 3 |
| 4개 이상 (복수 단원 복합) | 4 |
| 4개 이상 + 연결 방식 자체가 비자명 | 5 |

### 3-3. 풀이 단계 수 점수 (C)

| 풀이 단계 수 | 점수 |
|-------------|------|
| 1~2단계 | 1 |
| 3단계 | 2 |
| 4~5단계 | 3 |
| 6~8단계 | 4 |
| 9단계 이상 또는 단계 수 자체를 결정하기 어려움 | 5 |

### 3-4. 학교급 보정 계수 (S)

동일한 개념도 학교급·학년에 따라 상대적 난이도가 다릅니다.

| school_level | grade | 보정 방향 |
|-------------|-------|----------|
| middle | 1 | 원점수 그대로 또는 +0.3 (낮은 학년 기준 평가) |
| middle | 2 | 원점수 그대로 |
| middle | 3 | 원점수 그대로 또는 -0.3 (고등 준비 수준 포함) |
| high | 1 | 원점수 그대로 또는 +0.3 (수학 I·II 개념 도입기) |
| high | 2 | 원점수 그대로 |
| high | 3 | 원점수 그대로 또는 -0.5 (수능 수준 문제 상시 노출) |
| unknown / None | — | 보정 없이 원점수 사용 |

> 보정은 합산 점수 계산 후 최종 레벨 결정 단계에서 ±0.5 범위 내에서만 적용합니다.

### 3-5. 최종 레벨 결정

**합산 점수 = (A + B + C) / 3** (소수점 유지, 반올림 전)

| 합산 점수 범위 | 기본 레벨 |
|--------------|---------|
| 1.0 ~ 1.6 | 1 |
| 1.7 ~ 2.4 | 2 |
| 2.5 ~ 3.3 | 3 |
| 3.4 ~ 4.2 | 4 |
| 4.3 ~ 5.0 | 5 |

학교급 보정 적용 후 점수가 경계에 걸릴 경우, **행동영역 점수(A)를 우선 기준**으로 레벨을 결정합니다.

## 4. 판정 시 고려사항

### 4-1. 학교급·학년에 따른 상대 난이도

- **같은 개념, 다른 난이도**: "이차방정식 풀기"는 중2에서 level=2이지만, 고1에서 review 수준의 level=1일 수 있습니다.
- **grade=None 처리**: school_level이 주어진 경우 해당 학교급의 평균 학년(중학교→2학년, 고등학교→1학년)을 가정하여 판정합니다.
- **school_level=unknown**: 문제의 내용·어휘·기호로 중등/고등을 추정한 후, 추정 학교급을 명시하고 판정합니다.

### 4-2. 문제 유형별 고려

| type | 고려사항 |
|------|---------|
| multiple_choice | 선택지 자체가 힌트가 될 수 있어 실제 풀이 난이도보다 낮게 느껴질 수 있음. 선택지 오답이 정교할수록(근사값, 유사 표현) 난이도 상향 |
| short_answer | 정답 형태가 단순해도 풀이 과정이 복잡할 수 있음. 풀이 과정을 기준으로 판정 |
| essay | 논리적 서술·증명 요구 여부를 행동영역 판정에 반영 |

### 4-3. refer(출처) 활용 지침

출처가 명시된 경우 아래를 참고하되, 출처가 최종 판정을 결정하지는 않습니다. **문제 내용 자체를 우선합니다.**

| 출처 키워드 | 참고 난이도 범위 |
|------------|----------------|
| 개념기본서, 개념원리, 수학의 정석(기본편) | level 1~2 |
| RPM, 올림포스, 수학의 정석(실력편) | level 2~3 |
| 쎈(A/B), 마플시너지(기본) | level 2~3 |
| 쎈(C), 마플시너지(발전), 일품 | level 3~4 |
| 블랙라벨, 최상위, 킬러문항집 | level 4~5 |
| 내신 기출(중간/기말) 쉬운 문제 | level 1~2 |
| 내신 기출 서술형·고난도 | level 3~5 |
| 수능 2점 | level 1 |
| 수능 3점 | level 2~3 |
| 수능 4점 | level 3~4 |
| 수능 킬러(21·29·30번) | level 5 |

### 4-4. 경계값 판정 지침

- **level 2 vs 3 경계**: "조건 해석이 필요한가?"를 핵심 기준으로. 문제에서 요구하는 것이 무엇인지 한 번에 명확하면 level=2, 조건을 재해석하거나 식을 세우는 과정이 필요하면 level=3.
- **level 3 vs 4 경계**: "개념 연계가 3개를 넘어 단원 간 이동이 발생하는가?"를 확인. 단일 단원 내 심화면 level=3, 복수 단원 필수 결합이면 level=4.
- **level 4 vs 5 경계**: "풀이 경로를 발견하는 것 자체가 어려운가?" 표준 유형의 어려운 버전이면 level=4, 유형 자체가 생소하거나 아이디어 발상이 핵심이면 level=5.

## 5. 판정 흐름

LLM은 아래 단계를 순서대로 수행하여 최종 레벨을 출력합니다.

```
[Step 1] 입력 정보 파악
  - school_level, grade 확인
  - school_level=unknown이면 문제 내용으로 학교급 추정
  - grade=None이면 해당 학교급 평균 학년 가정

[Step 2] 개념 파악
  - 문제에서 요구하는 수학 개념을 열거 (최대 5개)
  - 예: ["이차방정식", "근과 계수의 관계", "부등식"]
  - 연계 개념 수 → B 점수 결정

[Step 3] 행동영역 판별
  - 아래 질문에 순서대로 답하여 A 점수 결정:
    Q1. 공식에 수를 직접 대입하거나 정의를 그대로 확인하는가? → Yes: A=1
    Q2. 표준 절차를 따르되 중간 계산이 필요한가? → Yes: A=2
    Q3. 조건 해석 또는 식 변형이 필요한가? → Yes: A=3
    Q4. 여러 중간 결과를 연결해 다음 단계를 도출하는가? → Yes: A=4
    Q5. 풀이 경로 자체를 발견하거나 고차 논증이 필요한가? → Yes: A=5

[Step 4] 풀이 단계 수 추정
  - 문제를 처음 접하는 평균적인 학생 기준으로 필요한 단계 수를 추정
  - C 점수 결정

[Step 5] 합산 점수 계산
  - Raw Score = (A + B + C) / 3

[Step 6] 학교급 보정 적용
  - 4절의 보정 계수(S)를 참고하여 ±0.5 이내에서 보정
  - 경계값 판정은 A 점수(행동영역)를 우선 기준으로 결정

[Step 7] 최종 레벨 출력
  - 레벨 (1~5 정수)
  - 판정 근거 요약 (1~2문장)
  - A, B, C 점수 및 Raw Score 명시
```

## 6. 레벨별 예시

아래 예시는 실제 DB의 문제 특성을 기반으로 구성된 대표 사례입니다.

---

### Level 1 예시

**문제:**
> 포물선 `y² = 8x`의 초점의 좌표를 구하시오.

- **school_level**: high, **grade**: 1
- **개념**: 포물선 표준형 `y² = 4px`에서 초점 `(p, 0)` 읽기
- **A(행동영역)**: 1 (정의 직접 대입)
- **B(개념 수)**: 1
- **C(풀이 단계)**: 1 (4p=8 → p=2 → 초점 (2,0))
- **Raw Score**: (1+1+1)/3 = 1.00 → **Level 1**

---

### Level 2 예시

**문제:**
> 서로 다른 5개의 책 중에서 3개를 골라 순서대로 나열하는 경우의 수를 구하시오.

- **school_level**: high, **grade**: 1
- **개념**: 순열 공식 `P(n,r) = n!/(n-r)!`
- **A**: 2 (공식 인식 + 대입 + 계산)
- **B**: 1 (순열 단일 개념)
- **C**: 2 (`P(5,3) = 5×4×3 = 60`, 2단계)
- **Raw Score**: (2+1+2)/3 = 1.67 → **Level 2**

---

### Level 3 예시

**문제:**
> 첫째항이 `a`, 공차가 `d`인 등차수열에서 세 번째 항은 7이고, 첫째항부터 다섯 번째 항까지의 합이 35일 때, `a`와 `d`를 구하시오.

- **school_level**: middle, **grade**: 3
- **개념**: 등차수열 일반항 + 등차수열 합 공식
- **A**: 3 (두 조건에서 연립방정식 설정 → 풀기)
- **B**: 2 (일반항, 합 공식)
- **C**: 3 (조건식 2개 설정 → 연립 → 해)
- **Raw Score**: (3+2+3)/3 = 2.67 → **Level 3**

---

### Level 4 예시

**문제:**
> 두 원 `x²+y²=4`와 `x²+y²-6x+8=0`의 두 교점을 지나는 직선과 원점 사이의 거리를 구하시오. 이 직선이 반지름 `r`인 원에 접할 때, `r`의 최솟값을 구하시오.

- **school_level**: high, **grade**: 2
- **개념**: 원의 방정식 → 두 원의 교점 → 공통현의 직선 방정식 → 점과 직선 사이의 거리
- **A**: 4 (복합 개념 연결, 중간 결과 활용)
- **B**: 4 (원의 방정식, 공통현, 거리 공식, 접선 조건)
- **C**: 5 (두 원 방정식 빼기 → 직선 방정식 → 원점까지 거리 → 접선 조건)
- **Raw Score**: (4+4+5)/3 = 4.33 → **Level 4** (경계: A=4 우선 적용)

---

### Level 5 예시

**문제:**
> 함수 `f(x) = log₂(x² - ax + b)` 가 정의되는 정수 `x`의 개수가 정확히 3개일 때, 정수 쌍 `(a, b)`의 개수를 구하시오. (단, `a, b`는 정수이고 `1 ≤ a ≤ 10`)

- **school_level**: high, **grade**: 2
- **개념**: 로그 정의역 조건(진수>0) → 이차부등식의 정수해 개수 분석 → 판별식·근의 위치 → 경우의 수 합산
- **A**: 5 (조건의 해석 자체가 비자명, 경우 분류 + 귀납적 합산)
- **B**: 5 (로그, 이차부등식, 판별식, 정수해, 경우의 수)
- **C**: 5 (경우 분류만 7~9가지 이상)
- **Raw Score**: (5+5+5)/3 = 5.00 → **Level 5**
