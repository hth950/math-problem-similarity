# 대한민국 중·고등 수학 문제 난이도 평가 가이드라인 (프롬프트 구성용)

## 📌 목적
이 문서는 LLM(대규모 언어 모델)을 활용하여 대한민국 중등 및 고등 수학 문제의 난이도를 1~5단계로 자동 분류하기 위한 기준을 정의합니다. 
이 기준을 바탕으로 LLM 프롬프트를 작성하여 입력된 문제 정보를 분석하고, 최종적으로 DB에 난이도 정보를 삽입하는 데 사용됩니다.

---

## 📥 LLM 입력 데이터 구조
LLM이 난이도를 정확하게 판별하기 위해 아래 4가지 요소가 종합적으로 제공되어야 합니다.

1. **문제 (Question Statement):** 문제의 발문. 사용된 어휘의 추상성, 조건의 개수 등을 파악.
2. **보기 (Conditions/Boxed Text):** <보기> 형태의 ㄱ, ㄴ, ㄷ 진위 판별이나 (가), (나) 등의 추가 조건. 조건의 복잡도와 결합 정도를 파악.
3. **선지 (Choices):** 객관식 1~5번 선지. 선지의 형태(단순 상수, 복잡한 식)에 따른 계산량 유추.
4. **해설 (Solution):** 정답을 도출하기 위한 논리적 전개 과정. (가장 중요한 평가 척도)
   - 개념의 수 (몇 개의 단원/개념이 융합되었는가?)
   - 풀이의 단계 (몇 Step에 걸쳐 풀어야 하는가?)
   - 케이스 분류 (경우의 수를 나누어 접근해야 하는가?)

---

## 📊 난이도 1~5단계 분류 기준

대한민국 수학 교육과정(수능/모의평가 및 내신 기준)을 반영한 1~5단계 세부 기준입니다.

| 난이도 | 명칭 | 대상 수준 (참고) | 개념 및 논리 전개 (해설 기준) | 문제/보기 형태 |
|:---:|:---|:---|:---|:---|
| **1** | **최하 (기초)** | 교과서 기본 예제<br>수능 2점 문항 | - 단일 기본 개념/공식의 직접 대입<br>- 풀이 과정이 1~2줄 이내로 매우 직관적임<br>- 함정이나 복잡한 계산이 없음 | 조건이 거의 없고 발문이 매우 짧음 ("~을 구하시오") |
| **2** | **하 (기본)** | 시중 교재(쎈 등) A/B단계 하<br>수능 3점 기본 문항 | - 단일 개념을 사용하나 기초적인 식 변형이 필요함<br>- 풀이 과정이 2~3단계 이내로 전개됨<br>- 전형적인 유형으로 계산 과정이 평이함 | 1~2개의 단순 조건이 주어지며, 명확한 수학적 용어가 사용됨 |
| **3** | **중 (보통)** | 시중 교재 보통/핵심 유형<br>수능 3점 상 ~ 4점 쉬운 문항 | - 2개 이상의 개념이 결합되거나, 약간의 응용력이 필요함<br>- 풀이 과정이 3~5단계 정도로 이루어짐<br>- 특정 조건에 따라 간단한 식 세우기가 요구됨 | 조건(가, 나)이 주어질 수 있으며, ㄱ, ㄴ, ㄷ 보기 중 평이한 수준의 추론 요구 |
| **4** | **상 (심화)** | 내신 변별력 문항<br>수능 4점 준킬러 (13~14, 20~21번 등) | - 3개 이상의 개념 융합 또는 단원 간 연계 발생<br>- 해설이 길며, 상황을 여러 케이스로 '분류'해야 풀 수 있음<br>- 높은 수준의 수식 조작이나 기하학적 직관(보조선 등) 요구 | 발문이 길거나, 추상적인 함수식/기호(새롭게 정의된 함수 등)가 등장함 |
| **5** | **최상 (초고난도)** | 최상위권용 심화 교재(블랙라벨 등)<br>수능 4점 킬러 (15, 22, 30번 등) | - 교육과정 내 개념을 극한으로 응용한 낯선 형태의 문제<br>- 매우 복잡한 케이스 분류와 깊은 논리적 추론(발상) 요구<br>- 풀이 과정이 매우 길고, 중간에 계산 실수 유발 요소가 많음 | 조건이 매우 까다롭고 추상적이며, 조건을 해석하는 데에만 상당한 시간이 소요됨 |

## 💡 LLM 프롬프트 템플릿 예시 (Prompt Template)

이 문서를 기반으로 LLM에 전달할 시스템 프롬프트(System Prompt) 구조 예시입니다.

```text
최고의 대한민국 중고등 수학 교육 전문가로서, 아래 주어진 수학 문제의 데이터(문제, 보기, 선지, 해설)를 분석하여 난이도를 1에서 5 사이의 정수로 평가해 주세요.

[난이도 판단 기준]
- 1 (최하): 단일 단순 공식 적용, 1~2줄 풀이. (교과서 예제)
- 2 (하): 기본 유형, 약간의 식 변형, 2~3단계 풀이. (일반 3점)
- 3 (중): 2개 이상 개념 결합, 3~5단계 풀이. (어려운 3점~쉬운 4점, 내신 빈출)
- 4 (상): 케이스 분류 필수, 복잡한 계산, 다중 개념 융합. (수능 준킬러)
- 5 (최상): 고도의 추론 및 낯선 상황 해석, 매우 긴 서술형 풀이 필요. (수능 킬러)

[추론 지침]
1. 해설(Solution)의 길이를 보고 논리(Step)의 깊이를 파악하세요.
2. 해설에 "경우를 나누면", "i), ii)"와 같은 케이스 분류가 많을수록 난이도는 최소 4 이상일 확률이 높습니다.
3. 보기(Conditions)에 (가), (나)와 같은 다중 복합 조건이나 새로운 기호 정의가 존재하면 4 또는 5점으로 가중치를 부여하세요.

[입력 데이터]
- 문제: {question_text}
- 보기: {box_conditions}
- 선지: {multiple_choices}
- 해설: {solution_text}

[출력 형식]
JSON 형식으로 응답해 주세요.
{{
  "reason": "난이도를 평가한 구체적인 논리적 근거 1~2줄",
  "difficulty": 3
}}
```
