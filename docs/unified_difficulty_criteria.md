# 수학 문제 난이도 통합 판정 기준서

## 문서 개요

### 목적

이 문서는 LLM이 대한민국 중등/고등 수학 문제의 난이도를 1~5등급으로 자동 판정하기 위한 **통합 기준**을 제공한다. 세 개의 기존 기준 문서(정량적 기준, 프롬프트 가이드, 상세 판정 매트릭스)를 하나로 통합하여, 사람이 읽기 위한 **기준 문서(Part 1)**와 LLM API 호출에 바로 사용할 수 있는 **프롬프트(Part 2)**로 구성한다.

### 대상 데이터

| 항목 | 내용 |
|------|------|
| 총 문제 수 | 약 100,000문제 |
| school_level | middle(35,384), high(62,937), elementary, unknown |
| grade | 1, 2, 3, None |
| type | multiple_choice(78,483), short_answer(21,498), essay(19) |
| 입력 필드 | question, refer, choice1~5, school_level, grade, answer, solution |
| 출력 필드 | difficulty (1~5 정수) |

## 1. 난이도 등급 정의

### Level 1 --- 기초 (Very Easy)

| 항목 | 기준 |
|------|------|
| **등급명** | 기초 |
| **한줄 정의** | 공식/정의를 직접 대입하거나 단순 계산으로 풀 수 있는 문제 |
| **행동영역** | 계산(Computation) / 지식 확인(Knowledge Recall) |
| **개념 수** | 1개, 단일 단원 |
| **풀이 단계** | 1~2단계 |
| **조건 복잡도** | 명시적 조건 1개, 함정 없음 |
| **핵심 발상** | 불필요 |
| **선지 난이도** | 단순, 오답 매력도 낮음 |
| **해설 길이** | 1~3줄 |
| **예상 정답률** | 85% 이상 |
| **수능 대응** | 2점 문항 (1~3번대) |

**교재 대응:**

| 학교급 | 대응 교재 |
|--------|-----------|
| 중등 | 교과서 기본, 개념+유형(개념편), 체크체크, 빨리 이해하는 수학 |
| 고등 | 교과서 기본, 수력충전, 50일 수학 |

**예시 특성:**
- 정의를 그대로 적용: "다음 중 소수인 것은?", 약수의 개수 구하기
- 기본 공식 대입: 일차방정식 풀기(이항 1회), 기본 도형 넓이/둘레
- 단순 값 대입: `f(x)=2x+1`에서 `f(3)` 구하기
- 정의 직접 확인: 포물선 초점 좌표 구하기, `sin30` 값
- 그래프 읽기, 표에서 값 찾기

### Level 2 --- 기본 (Easy)

| 항목 | 기준 |
|------|------|
| **등급명** | 기본 |
| **한줄 정의** | 기본 유형을 인식하고 표준 풀이 절차를 적용하는 문제 |
| **행동영역** | 이해(Comprehension) / 기본 적용(Basic Application) |
| **개념 수** | 2~3개, 주로 단일 단원 내 |
| **풀이 단계** | 2~4단계 |
| **조건 복잡도** | 명시적 조건 2~3개, 직관적 파악 가능 |
| **핵심 발상** | 불필요 |
| **선지 난이도** | 약간 유사, 오답 매력도 약간 존재 |
| **해설 길이** | 3~6줄 |
| **예상 정답률** | 65~85% |
| **수능 대응** | 쉬운 3점 문항 |

**교재 대응:**

| 학교급 | 대응 교재 |
|--------|-----------|
| 중등 | 개념쎈, 라이트쎈, 우공비Q 표준편 |
| 고등 | RPM, 라이트쎈, 개념원리RPM |

**예시 특성:**
- 연립방정식 풀기(가감법/대입법 1회)
- 이차방정식 인수분해로 풀기
- 순열/조합 공식 적용 (단순 경우 수 계산)
- 공식을 알면 바로 풀리지만, 어떤 공식을 쓸지 판단 필요
- 조건에서 변수를 구한 뒤 대입하는 2단계 문제

### Level 3 --- 표준 (Medium)

| 항목 | 기준 |
|------|------|
| **등급명** | 표준 |
| **한줄 정의** | 2~3개 개념을 결합하여 표준 유형을 풀되, 약간의 응용/변형이 필요한 문제 |
| **행동영역** | 적용(Application) / 분석(Analysis) 초입 |
| **개념 수** | 3~4개, 단원 내 혹은 인접 단원 간 |
| **풀이 단계** | 4~6단계 |
| **조건 복잡도** | 숨겨진 조건 있음, 분석력 필요 |
| **핵심 발상** | 약간 필요 |
| **선지 난이도** | 함정 선지 포함, 오답 매력도 높음 |
| **해설 길이** | 6~10줄 |
| **예상 정답률** | 40~65% |
| **수능 대응** | 어려운 3점 ~ 쉬운 4점 |

**교재 대응:**

| 학교급 | 대응 교재 |
|--------|-----------|
| 중등 | 쎈 B~C, 유형아작, 개념+유형(파워편), 최상위라이트 |
| 고등 | 쎈 C, 일품, 고쟁이 |

**예시 특성:**
- 이차방정식 + 도형: 변수로 놓고 이차방정식 세우기
- 연립방정식 활용문제: 속도/거리/시간, 농도 혼합
- 함수 그래프 성질 + 미분 조건 결합
- 조건부 확률 + 경우의 수 결합
- ㄱ, ㄴ, ㄷ 합답형 중 비교적 평이한 수준

> **주의:** 기존 데이터에서 전체 문제의 약 34%가 이 레벨에 집중되어 있다. 경계 판정 시 Level 2, Level 4와 명확히 구분해야 한다.

| 항목 | 기준 |
|------|------|
| **등급명** | 심화 |
| **한줄 정의** | 복합 개념을 연결하고 다단계 추론이 필요한 문제 |
| **행동영역** | 분석(Analysis) / 종합(Synthesis) |
| **개념 수** | 4~5개 이상, 단원 간 융합 |
| **풀이 단계** | 6~10단계 |
| **조건 복잡도** | 다중 조건 해석, 핵심 아이디어(접근법) 발견이 관건 |
| **핵심 발상** | 핵심 (비자명한 발상 필요) |
| **선지 난이도** | 정교한 함정, 세밀한 분석/반례 구성 필요 |
| **해설 길이** | 10~20줄 |
| **예상 정답률** | 15~40% |
| **수능 대응** | 어려운 4점 (준킬러, 18~22번 영역) |

**교재 대응:**

| 학교급 | 대응 교재 |
|--------|-----------|
| 중등 | 일등급, 최상위, 블랙라벨, 에이급 수학 A단계 |
| 고등 | 블랙라벨, 일등급 수학 |

**예시 특성:**
- 두 원의 교점과 제3의 원 사이 관계
- 적분 + 넓이 + 최솟값: `f(t) = integral |g(x)| dx` 형태
- 수열의 귀납적 정의에서 패턴 발견 후 일반항 구하기
- 정수론적 성질(모듈러 연산) + 수열 결합
- ㄱ, ㄴ, ㄷ 합답형 중 난이도 높은 유형

| 항목 | 기준 |
|------|------|
| **등급명** | 최고난도 |
| **한줄 정의** | 창의적 접근/고차원 추론이 요구되며, 풀이 경로 발견 자체가 핵심인 문제 |

| **선지 난이도** | 극정교, 정교한 반례 구성/극한 논증 필요 |

| **예상 정답률** | 15% 미만 (수능 킬러는 1~5%) |

**교재 대응:**

| 학교급 | 대응 교재 |
|--------|-----------|
| 중등 | 에이급 S단계, 슈퍼수학, 경시대회 |
| 고등 | 수능 킬러, 드릴, 수학의 신 최상위 |

**예시 특성:**
- 로그 부등식 + 함수 개수 판별 (보기 T/F 각각 판단)

- 미적분 + 도형 + 최솟값의 최솟값 (2중 최적화)
- 수열 + 점화식 + 귀납 증명 복합
- 대학 학부 수준 개념을 간접적으로 활용한 문제

## 2. 6축 판정 요소 매트릭스

문제의 난이도를 정량적으로 판정하기 위한 6가지 판정 요소와 점수 기준이다.

| 판정 요소 | 1점 (기초) | 2점 (기본) | 3점 (표준) | 4점 (심화) | 5점 (최고난도) |
|-----------|-----------|-----------|-----------|-----------|--------------|
| **개념 수** | 1개 | 2~3개 | 3~4개 | 4~5개+ (융합) | 5개+ (상위과정) |
| **풀이 단계** | 1~2 | 2~4 | 4~6 | 6~10 | 10+ |
| **조건 복잡도** | 명시적 1개 | 명시적 2~3개 | 숨겨진 조건 有 | 다중 조건 해석 | 다단계 논리 추론 |
| **핵심 발상** | 불필요 | 불필요 | 약간 필요 | 핵심 | 핵심 (고난도) |
| **선지 난이도** | 단순 | 약간 유사 | 함정 有 | 정교한 함정 | 극정교 / 반례 필요 |
| **해설 길이** | 1~3줄 | 3~6줄 | 6~10줄 | 10~20줄 | 20줄+ |

### 보조 참조 지표

| 지표 | 1 | 2 | 3 | 4 | 5 |
|------|---|---|---|---|---|
| **수능 배점** | 2점 | 쉬운 3점 | 어려운 3점 ~ 쉬운 4점 | 어려운 4점 (준킬러) | 킬러 (15,22,30번) |
| **예상 정답률** | 85%+ | 65~85% | 40~65% | 15~40% | 15% 미만 |

## 3. 판정 흐름 (6단계)

```
[Step 1] 입력 정보 파악
  - school_level, grade 확인
  - school_level=unknown이면 문제 내용으로 학교급 추정
  - grade=None이면 해당 학교급 평균 학년 가정 (중등→2학년, 고등→1학년)

[Step 2] 핵심 개념 식별
  - 문제에서 요구하는 수학 개념을 열거 (최대 5개)
  - 개념 수 → 판정 요소 ① 점수 결정

[Step 3] 풀이 단계 추정
  - 해설이 있으면 해설 기반, 없으면 문제 분석 기반
  - 풀이 단계 수 → 판정 요소 ② 점수 결정

[Step 4] 핵심 발상 필요 여부 판단
  - 비자명한 아이디어/접근법이 필요한가?
  - 조건의 복잡도 분석 → 판정 요소 ③, ④ 점수 결정

[Step 5] 선지/보기/해설 복잡도 평가
  - 선지의 함정/오답 매력도 → 판정 요소 ⑤ 점수 결정
  - 해설의 길이/논리 전개 복잡도 → 판정 요소 ⑥ 점수 결정

[Step 6] 매트릭스 종합 판정
  - 6개 요소 점수의 가중 평균 계산
  - 가중치: 개념수(1.0), 풀이단계(1.5), 조건복잡도(1.0), 핵심발상(1.5), 선지난이도(0.5), 해설길이(0.5)
  - 가중 평균 = (①x1.0 + ②x1.5 + ③x1.0 + ④x1.5 + ⑤x0.5 + ⑥x0.5) / 6.0
  - 경계값 판정 시 핵심 발상(④)과 풀이 단계(②)를 우선 기준으로 결정
```

### 가중 평균 → 레벨 매핑

| 가중 평균 범위 | 기본 레벨 |
|--------------|---------|
| 1.0 ~ 1.6 | 1 |
| 1.7 ~ 2.4 | 2 |
| 2.5 ~ 3.3 | 3 |
| 3.4 ~ 4.2 | 4 |
| 4.3 ~ 5.0 | 5 |

## 4. 학교급 보정 및 상대 난이도

### 핵심 원칙

중등과 고등은 같은 난이도 숫자라도 **절대적 난이도가 다르다**. 각 학교급 내에서의 **상대적 난이도**로 판정한다.

### 학교급/학년별 보정 방향

| school_level | grade | 보정 방향 |
|-------------|-------|----------|
| middle | 1 | 원점수 그대로 또는 +0.3 (낮은 학년 기준) |
| middle | 2 | 원점수 그대로 |
| middle | 3 | 원점수 그대로 또는 -0.3 (고등 준비 수준 포함) |
| high | 1 | 원점수 그대로 또는 +0.3 (수학 I/II 개념 도입기) |
| high | 2 | 원점수 그대로 |
| high | 3 | 원점수 그대로 또는 -0.5 (수능 수준 문제 상시 노출) |
| unknown / None | --- | 보정 없이 원점수 사용 |

> 보정은 가중 평균 계산 후 최종 레벨 결정 단계에서 +-0.5 범위 내에서만 적용한다.

### 중등/고등 교재 대응표

| 난이도 | 중등 대응 | 고등 대응 |
|--------|-----------|-----------|
| 1 | 교과서 기본 / 개념+유형(개념편) | 교과서 기본 / 수력충전 / 50일 수학 |
| 2 | 개념쎈, 라이트쎈 | RPM, 라이트쎈, 개념원리RPM |
| 3 | 쎈 B~C, 유형아작, 최상위라이트 | 쎈 C, 일품, 고쟁이 |
| 4 | 일등급, 최상위, 블랙라벨 | 블랙라벨, 일등급 수학 |
| 5 | 에이급 S단계, 슈퍼수학, 경시대회 | 수능 킬러, 드릴, 수학의 신 |

## 5. 경계값 판정 지침

### Level 2 vs 3 경계

- **핵심 질문:** "조건 해석이 필요한가?"
- 문제에서 요구하는 것이 한 번에 명확하면 → Level 2
- 조건을 재해석하거나 식을 세우는 과정이 필요하면 → Level 3
- 해설에서 개념 간 연결이 2회 이상이면 → Level 3

### Level 3 vs 4 경계

- **핵심 질문:** "개념 연계가 단원 간 이동을 수반하는가?"
- 단일 단원 내 심화면 → Level 3
- 복수 단원 필수 결합 + 케이스 분류 발생 → Level 4
- 해설에 "경우를 나누면", "i), ii)" 같은 케이스 분류가 등장 → 최소 Level 4

### Level 4 vs 5 경계

- **핵심 질문:** "풀이 경로를 발견하는 것 자체가 어려운가?"
- 표준 유형의 어려운 버전이면 → Level 4
- 유형 자체가 생소하거나 아이디어 발상이 핵심 → Level 5
- (가)(나) 다중 복합 조건 + 새로운 기호 정의 → Level 4~5

### Level 3 쏠림 방지

기존 데이터에서 Level 3에 34.4%가 집중되어 있으므로, 다음을 반드시 점검한다:
1. 숨겨진 조건이 **없으면** Level 2 검토
2. 핵심 발상이 **필요하면** Level 4 검토
3. 경계에서는 6축 매트릭스 점수를 개별 확인하여 차별화 근거를 명시

---

## 6. 출처(refer) 참고 지침

출처가 명시된 경우 아래를 **참고**하되, 출처가 최종 판정을 결정하지는 않는다. **문제 내용 자체를 우선한다.**

| 출처 키워드 | 참고 난이도 범위 |
|------------|----------------|
| 개념기본서, 개념원리, 수학의 정석(기본편) | 1~2 |
| RPM, 올림포스, 수학의 정석(실력편) | 2~3 |
| 쎈(A/B), 마플시너지(기본) | 2~3 |
| 쎈(C), 마플시너지(발전), 일품 | 3~4 |
| 블랙라벨, 최상위, 킬러문항집 | 4~5 |
| 내신 기출(중간/기말) 쉬운 문제 | 1~2 |
| 내신 기출 서술형/고난도 | 3~5 |
| 수능 2점 | 1 |
| 수능 3점 | 2~3 |
| 수능 4점 | 3~4 |
| 수능 킬러(21/29/30번) | 5 |

## 7. 문제 유형별 고려사항

| type | 고려사항 |
|------|---------|
| multiple_choice | 선택지 자체가 힌트가 될 수 있어 풀이 난이도보다 낮게 느껴질 수 있음. 선택지 오답이 정교할수록(근사값, 유사 표현) 난이도 상향 |
| short_answer | 정답 형태가 단순해도 풀이 과정이 복잡할 수 있음. 풀이 과정 기준으로 판정 |
| essay | 논리적 서술/증명 요구 여부를 행동영역 판정에 반영 |

## 8. 해설(Solution) 중심 판정 원칙

해설은 난이도 판정에서 **가장 중요한 평가 척도**이다.

### 해설 분석 포인트

1. **개념의 수**: 해설에서 사용된 수학 개념/공식이 몇 개인가?
2. **풀이의 단계**: 해설이 몇 Step으로 전개되는가?
3. **케이스 분류**: 경우의 수를 나누어 접근하는가?
4. **고급 기법**: 역함수, 합성함수, 귀납법 등 고급 기법이 사용되는가?

### 해설 기반 추론 규칙

| 해설 특성 | 추론 |
|-----------|------|
| 1~3줄, 공식 대입만 | Level 1 |
| 3~6줄, 기본 식 변형 | Level 2 |
| 6~10줄, 개념 연결 2~3회 | Level 3 |
| 10~20줄, 케이스 분류 등장 | Level 4 이상 |
| 20줄+, 여러 풀이법 존재, 고급 사고 필요 | Level 5 |
| "경우를 나누면", "i), ii)" 등 분류 표현 | 최소 Level 4 |
| (가)(나) 다중 복합 조건 + 새 기호 정의 | Level 4~5 |

# Part 2: LLM 프롬프트

---

## System Prompt

아래 텍스트를 System Prompt로 사용한다.

```text
당신은 대한민국 중등/고등 수학 문제의 난이도를 1~5단계로 판정하는 전문가입니다.

# 역할
주어진 수학 문제의 데이터(문제, 보기, 선지, 해설, 학교급, 학년)를 분석하여 난이도를 1~5 정수로 평가합니다.

# 난이도 등급 정의
- 1 (기초): 단일 개념/공식 직접 적용. 1~2단계 풀이. 정답률 85%+. 교과서 기본 예제, 수능 2점.
- 2 (기본): 2~3개 개념 결합, 기초적 식 변형. 2~4단계 풀이. 정답률 65~85%. 쉬운 3점.
- 3 (표준): 3개+ 개념 복합, 숨겨진 조건 파악 필요. 4~6단계 풀이. 정답률 40~65%. 어려운 3점~쉬운 4점.
- 4 (심화/준킬러): 다단원 융합, 케이스 분류 필수, 핵심 발상 필요. 6~10단계. 정답률 15~40%. 어려운 4점.
- 5 (최고난도/킬러): 상위과정 간접 차용, 접근 자체가 난해, 매우 긴 풀이. 10+단계. 정답률 15% 미만. 킬러(15,22,30번).

# 6가지 판정 요소 점수 매트릭스
각 요소를 1~5점으로 채점하세요.

1) 개념 수: 1개→1 | 2~3개→2 | 3~4개→3 | 4~5개+(융합)→4 | 5개+(상위과정)→5
2) 풀이 단계: 1~2→1 | 2~4→2 | 4~6→3 | 6~10→4 | 10+→5
3) 조건 복잡도: 명시적1개→1 | 명시적2~3개→2 | 숨겨진조건有→3 | 다중조건해석→4 | 다단계논리추론→5
4) 핵심 발상: 불필요→1 | 불필요→2 | 약간필요→3 | 핵심→4 | 핵심(고난도)→5
5) 선지 난이도: 단순→1 | 약간유사→2 | 함정有→3 | 정교한함정→4 | 극정교/반례필요→5
6) 해설 길이: 1~3줄→1 | 3~6줄→2 | 6~10줄→3 | 10~20줄→4 | 20줄+→5

# 판정 흐름
1. 문제에서 사용되는 핵심 개념을 식별한다.
2. 풀이에 필요한 단계 수를 추정한다.
3. 핵심 아이디어(비자명한 발상)의 필요 여부를 판단한다.
4. 선지/보기의 복잡도를 평가한다.
5. 해설의 길이와 논리 전개 복잡도를 확인한다.
6. 6가지 요소 점수를 종합하여 최종 난이도를 결정한다.

# 핵심 추론 지침
1. 해설(solution)이 있으면 해설을 최우선 판정 근거로 삼을 것. 해설의 길이와 논리 깊이가 가장 신뢰도 높은 지표이다.
2. 해설이 없으면 문제+선지+보기만으로 판정하되, confidence를 "low"로 표기할 것.
3. 해설에 "경우를 나누면", "i), ii)", "Case 1, Case 2" 같은 케이스 분류가 등장하면 최소 4 이상이다.
4. 보기에 (가), (나) 같은 다중 복합 조건이나 새로운 기호 정의가 있으면 4~5로 가중치를 부여하라.
5. 중등과 고등은 같은 난이도 숫자라도 절대적 수준이 다르다. 각 학교급 내에서의 상대적 난이도로 판정하라.
6. Level 3에 과도하게 몰리지 않도록, 경계에서는 차별화 근거를 반드시 명시하라.
   - 숨겨진 조건이 없으면 → Level 2 검토
   - 핵심 발상이 필요하면 → Level 4 검토
7. 단답형(short_answer) 문제는 정답이 단순해도 풀이 과정 기준으로 판정하라.

# 경계값 판정 기준
- Level 2 vs 3: "조건 해석이 필요한가?" — 한 번에 명확하면 2, 재해석/식세우기 필요시 3
- Level 3 vs 4: "개념 연계가 단원 간 이동을 수반하는가?" — 단일 단원 심화 3, 복수 단원 결합 4
- Level 4 vs 5: "풀이 경로 발견 자체가 어려운가?" — 어려운 표준유형 4, 생소한 유형/발상 핵심 5

# 교재 대응 참고
[중등] 1:교과서/개념+유형 | 2:개념쎈/라이트쎈 | 3:쎈B~C/유형아작 | 4:일등급/최상위/블랙라벨 | 5:에이급S/경시대회
[고등] 1:교과서/수력충전 | 2:RPM/라이트쎈 | 3:쎈C/일품/고쟁이 | 4:블랙라벨/일등급 | 5:수능킬러/드릴/수학의신

# 출력 형식
반드시 아래 JSON 형식으로만 응답하세요. 다른 텍스트를 포함하지 마세요.
{
  "difficulty": <1~5 정수>,
  "reason": "<판정 근거 2~3문장>",
  "confidence": "<high|medium|low>",
  "scores": {
    "concept_count": <1~5>,
    "steps": <1~5>,
    "condition_complexity": <1~5>,
    "key_insight": <1~5>,
    "choice_difficulty": <1~5>,
    "solution_length": <1~5>
  }
}
```

## User Prompt 템플릿

아래 텍스트를 User Prompt 템플릿으로 사용한다. `{...}` 부분에 실제 데이터를 삽입한다.

```text
아래 수학 문제의 난이도를 판정해 주세요.

[메타 정보]
- 학교급(school_level): {school_level}
- 학년(grade): {grade}
- 문제 유형(type): {type}

[문제]
{question}

[보기/참고]
{refer}

[선지]
1) {choice1}
2) {choice2}
3) {choice3}
4) {choice4}
5) {choice5}

[해설]
{solution}
```

### 필드별 처리 규칙

| 필드 | 필수 여부 | 없을 때 처리 |
|------|----------|-------------|
| school_level | 필수 | unknown으로 전달 → LLM이 문제 내용으로 추정 |
| grade | 필수 | None으로 전달 → LLM이 학교급 평균 학년 가정 |
| type | 선택 | 생략 가능 |
| question | 필수 | 판정 불가 |
| refer | 선택 | 없으면 "없음" 또는 빈 문자열 |
| choice1~5 | 선택 | 단답형/서술형이면 "없음" 또는 빈 문자열 |
| solution | 선택 | 없으면 "없음" → confidence가 "low"로 떨어짐 |

## Solution이 없는 경우 Fallback

solution이 없는 문제는 전체의 일부를 차지한다. 이 경우에도 판정은 가능하지만 신뢰도가 낮아진다.

**Fallback 전략:**
1. 문제 텍스트의 조건 수, 요구하는 행동(구하시오/증명하시오/판별하시오)으로 행동영역 추정
2. 선지의 복잡도(단순 숫자 vs 복잡한 식)로 풀이 단계 추정
3. 출처(refer)가 있으면 참고 난이도 범위와 교차 검증
4. confidence를 반드시 "low"로 설정
